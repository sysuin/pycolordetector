Introduction
============

Description
-----------
**pycolordetector** is a Python package for detecting colors in images. It provides fast and easy way to detect color of a pixel. The applications build through this package will automatically give the name of the color just by double-clicking on them. pycolordetector has been designed to make working with image data both simple and intuitive. It aims to be the fundamental high-level building block for doing practical and real-world analysis on images.Â

-----------
